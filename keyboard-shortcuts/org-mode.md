## Org Mode

```
+------------------------------+------------------------------------------------+
| Keys                         | Function                                       |
+------------------------------+------------------------------------------------+
| M-F5                         | to open emacs menu                             |
| TAB                          | To expand/close headline                       |
| C-c C-t                      | Switch between TODO/DONE                       |
| M-shift-return               | Insert new outline item                        |
| [[link][description]]        | hyperlink                                      |
| C-c C-l                      | To insert link instead of typing               |
| C-c C-o                      | to open a link                                 |
| Shift-Tab                    | Overview - outline headings only               |
| C-c C-n                      | Next headline                                  |
| C-c C-p                      | previous headline                              |
| M-up arrow                   | Move/reorder heading up                        |
| M-down arrow                 | Move/reorder heading down                      |
| M-shift-left arrow           | outent heading                                 |
| M-shift-right arrow          | indent heading                                 |
| - or 1)                      | bulleted/numbered list                         |
| - [ ]                        | check for task                                 |
| C-c C-c                      | toggle checkbox  / also align table            |
| - <text> ::                  | Description list                               |
| ESC-u                        | upper case word                                |
| ESC-l                        | lower case                                     |
| #+begin-example +end_example | source code, unformatted                       |
| C-c C-e                      | export file                                    |
| C-c -                        | dash line in table                             |
| M-`                          | Menu                                           |
| M-` T                        | Table menu                                     |
| :=                           | Formula eg., :=$2*$3 = col 2*3                 |
| C-u C-c C-l                  | link a file                                    |
| C-x k                        | Kill buffer                                    |
| C-x 0                        | single window - current buffer                 |
| C-x 1                        | single window - mini buffer file               |
| C-x 2                        | split window vertical                          |
| C-x 3                        | split window horizontal                        |
|                              |                                                |
| C-c C-s                      | Schedule task to a date                        |
| C-c C-d                      | Set deadline                                   |
| C-x n e                      | narrow to element - show only subtree          |
| C-x n w                      | widen                                          |
+------------------------------+------------------------------------------------+
| m                            | mark                                           |
| u                            | unmark                                         |
| B $                          | archive selected items                         |
+------------------------------+------------------------------------------------+
| C-c C-x ;                    | set a timer                                    |
| C-c C-x C-w                  | to cut heading with sub-tree (org-cut-special) |
| C-c C-x C-y                  | to paste cut (org-paste-special)               |
| C-c C-w                      | org-refile                                     |
| C-x C-+ and C-x C--          | Font size increase/decrease                    |
+------------------------------+------------------------------------------------+

```
